# 操作台线框变更

基线：`docs/wireframes/pages/console.md`

## pages/console.md

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│ ● ● ●  [侧栏]                                                               │
│ ┌──────────────────┬─────────────────────────────────────────────────────────┤
│ │ Moebius       搜索│                                                         │
│ │ ＋ 新会话         │                                                         │
│ │                  │                    多 agent 共用的                       │
│ │ 项目             │                       对话流                            │
│ │ ▾ agent-moebius  │                                                         │
│ │   当前会话       │      空态仅显示提示；有消息后显示连续时间线             │
│ │   裂变会话 A     │                                                         │
│ │   裂变会话 B     │                                                         │
│ │   失败状态       │                                                         │
│ │   等待确认       │                                                         │
│ │                  │       ┌──────────────────────────────────────┐          │
│ │                  │       │ agent-moebius  本地  main            │          │
│ │                  │       ├──────────────────────────────────────┤          │
│ │                  │       │ 描述目标，@ 一个角色…            ↑ │          │
│ │ 用户          帮助│       └──────────────────────────────────────┘          │
│ └──────────────────┴─────────────────────────────────────────────────────────┘
└──────────────────────────────────────────────────────────────────────────────┘
```

要求：
- 窗口控制与侧栏融合，不保留独立白色标题条。
- 不保留会话统计顶部工具栏；状态进入侧栏行或时间线。
- 所有 agent 消息共用一条对话流，以头像、名称和状态区分。
- 工作区模式和分支位于输入器上下文行；原始机器数据只在开发者诊断中出现。
- 会话在运行时可以裂变，但侧栏始终平铺为同级列表，不展示树线、缩进或展开层级。
- 空会话、普通对话、运行中、等待真人、失败/卡住和裂变会话列表共享同一页面骨架。
