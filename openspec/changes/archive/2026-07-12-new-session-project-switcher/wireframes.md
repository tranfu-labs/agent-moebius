# Wireframes：new-session-project-switcher

Baseline: `docs/wireframes/pages/console.md`.

## pages/console.md

### 项目行新建与空白会话项目菜单

```text
┌────────────────────┬───────────────────────────────────────────────────────┐
│ ● ● ●  Moebius     │                                                       │
│ ▱ 打开项目          │                                                       │
│                    │                                                       │
│ 项目               │                                                       │
│ ▱ moebius  ＋ │                    空白新会话                         │
│   新会话            │                                                       │
│   本地 T4 验收      │                                                       │
│ ▱ demo-project   ＋ │                                                       │
│   设计讨论          │                                                       │
│                    │       ┌──────────────────────────────────────┐        │
│                    │       │ [▱ moebius⌄] [本地] [当前分支] │        │
│                    │       │ 描述你的目标…                    ↑  │        │
│                    │       └──────────────────────────────────────┘        │
└────────────────────┴───────────────────────────────────────────────────────┘
```

Requirements:
- 每个 `＋` 只为同一行项目创建并选中新会话。
- 全局“新会话”行不再出现，避免隐式项目归属。
- 空白新会话的文件夹上下文是项目菜单触发器；workspace 与 branch 上下文保持原位。

### 项目菜单展开态

```text
                              ┌────────────────────────┐
                              │ ✓ moebius        │
                              │   demo-project         │
                              └────────────────────────┘
       ┌──────────────────────────────────────────────┐
       │ [▱ moebius⌃] [本地] [当前分支]         │
       │ 未发送的输入草稿                         ↑  │
       └──────────────────────────────────────────────┘
```

Requirements:
- 菜单列出所有已打开项目并标记当前项目。
- 选择另一项目后，同一空白 session 移入目标项目，草稿与选中态保持。
- 菜单支持键盘导航、退出与焦点回到触发器。
- 重绑请求到新状态提交期间，菜单触发器与发送动作同时禁用；失败后恢复原项目、原草稿与交互能力。

### 历史会话项目锁定态

```text
       ┌──────────────────────────────────────────────┐
       │ ▱ moebius  [本地] [当前分支]           │
       │ 描述你的目标…                            ↑  │
       └──────────────────────────────────────────────┘
```

Requirements:
- 已有消息、运行记录或 parent/child 关系时不显示下拉箭头或项目菜单。
- 项目名称仍可见，但不能迁移历史会话。
