# 字符图：add-desktop-shell

基线说明：本 change 新增桌面状态页，为全新页面，无既有基线；observer 页版式不变（其基线为 `docs/wireframes/pages/observer.md`，本 change 不触碰）。

## pages/desktop-status.md

桌面应用主窗口（状态指示页）。只读呈现 + 三个动作（打开观察页 / 打开数据目录 / 检查更新），无任何配置写入口。

正常态：

```text
┌──────────────────────────────────────────────┐
│  agent-moebius                       v0.1.0  │
├──────────────────────────────────────────────┤
│  运行状态                                     │
│   ● runner    运行中                          │
│   ● observer  127.0.0.1:52341   [打开观察页]  │
│                                              │
│  环境自检                                      │
│   ✓ codex CLI   已找到                        │
│   ✓ gh CLI      已登录 (aquarius-wing)        │
│   ⚠ 仓库白名单   未配置        [打开数据目录]   │
│                                              │
├──────────────────────────────────────────────┤
│  当前版本 0.1.0                    [检查更新]  │
└──────────────────────────────────────────────┘
```

崩溃重启态（runner 行替换）：

```text
│   ◐ runner    已崩溃，第 2/3 次重启中…         │
```

连续崩溃停住态（runner 行替换，附日志位置）：

```text
│   ✗ runner    已停止（连续崩溃 3 次）           │
│               日志：~/.agent-moebius/logs/…   │
```

自检异常态（对应行替换）：

```text
│   ✗ codex CLI   未找到，请安装后重启应用        │
│   ✗ gh CLI      未登录，请运行 gh auth login   │
```

流转说明（归档时同步进 `docs/wireframes/flow.md`）：

```text
启动应用
  ├─ 主窗口打开 → 状态页接收主进程快照推送（运行状态 / 自检 / 版本）
  ├─ [打开观察页] → 默认浏览器打开 observer 动态端口地址
  ├─ [打开数据目录] → 系统文件管理器打开 ~/.agent-moebius
  ├─ [检查更新] → macOS：有新版跳转下载页；Windows/Linux：自动下载安装
  └─ 关闭窗口 → 停 runner → 关 observer → 应用退出
```
