# 设计：rack-badge-arc-fluidity

## 核心原则：丝滑 = 速度连续，叙事画进路径形状而非时间轴
上一版的教训：任何「中途所有运动分量同时归零再重启」的编排，播放时必然读作顿挫，与每一帧构图多好看无关。本版把「拿起再飞」从时间分拍改写为**空间弧线 + 分轴并行平滑曲线**。

## 幽灵卡：三层 wrapper 分轴驱动
```
.fly-layer（perspective 1500，origin 按 .rack 实测）
  .morph-ghost   —— X 平移（translateX + 居中修正），单条连续缓动，全程不停
    .mg-y        —— Y 平移（translateY + 居中修正），抛物弧
      .mg-spin   —— rotateY + scale，前重单曲线
        B（完整工牌克隆，铺满）
        A（架上简化卡面克隆，宽对齐居中，上层）
```
三层皆 `transform-style: preserve-3d`，让内层 rotateY 吃到 fly-layer 的透视。末态：X=工牌中心-50%宽、Y=工牌中心-50%高、spin=identity → 幽灵盒等于真工牌矩形，像素级换装不变。

各轴曲线（打开 600ms / 关闭 380ms，单段）：
- **X**：打开 `cubic-bezier(0.2,0,0,1)`；关闭 `cubic-bezier(0.4,0,0.2,1)`（加速离窗、减速入槽）。这是丝滑底线——水平速度全程 > 0。
- **Y**：三关键帧 `起点 → 弧顶(offset≈0.45/0.5) → 终点`，弧顶 = min(两端 y) − 40px（关 30px）。升段 `cubic-bezier(0.25,0,0.3,1)` 减速到顶（自然零速顶点），落段 `cubic-bezier(0.35,0,0.55,1)` 由零起速缓落。垂直零速只发生在物理自然的抛物线顶点，且与持续前进的水平分量叠加，无顿挫感。
- **spin**：打开 `cubic-bezier(0.05,0.7,0.1,1)`（前重，转正在弧线前段完成——上一版它做位移缓动嫌太陡，做旋转缓动恰好）；关闭 `cubic-bezier(0.7,0,0.85,0.5)`（后重，临近插槽才转回侧立）。

## 淡化：全部 ease-in-out
linear+硬拐点的透明度变化率突变（突然开始/停止淡）是次级闪动源。所有 opacity/filter 动画的分段缓动统一 `ease-in-out`——变化率始末为零。A/B 交叉淡化同窗口同曲线，互补和恒为 1。窗口：打开 A→B 25–60%、关闭 B→A 35–70%；头像收敛滑动同窗口。

## 变形面与真面板
- 变形面：打开延迟 12% 后单曲线从足迹长成面板（**起步零速自然，中途不停即可**），淡入 12–50%；关闭无延迟单曲线缩回，55–85% 淡出。
- 真面板：不 transform；内容打开 50–92% 淡入、关闭 0–22% 淡出。

## 遮罩 blur 挪出动画期（保帧率）
`backdrop-filter: blur(6px)` 在动画期间每帧重合成整个背景，是掉帧嫌疑。改为：变形期间遮罩纯色压暗（rgba 不变）；落定后给弹窗加 `.is-settled` 类，blur 经 250ms transition 渐入；关闭开始即移除类。降级路径打开时同步加类。

## 验证口径（本次升级的关键）
1. **速度采样（新增，主判据）**：真实播放中每帧（rAF）采样幽灵中心坐标，计算相邻水平位移增量序列；取中段（15%–85%）最小瞬时速度与峰值之比，MUST 显著大于零（阈值 ~10%）——上一版两拍在拼接点该比值≈0。
2. 落点对齐断言（沿用，四维差 ≤1px）；状态/防重入/焦点断言（沿用）。
3. 关键帧截图目检（辅助，不再作为运动质感的依据）。
