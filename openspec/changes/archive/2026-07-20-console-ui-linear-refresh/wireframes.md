# console-ui-linear-refresh 字符图

本 change 只改变 console 页的视觉版式（行结构、状态形态、属性面板头），不改变页面骨架与流转。基线引用 `docs/wireframes/pages/console.md`（「Codex-native Stream Anchor」节与「Running」节）；flow.md 无变化。

## pages/console.md

Requirements:
- 时间线是发丝线分隔的 Linear inbox 行序列，不是浮动卡片堆叠。
- 每条 agent 行：圆形角色头像右下角挂 stage 角标；行内为角色名（wght 510）+ stage（muted）+ 右侧状态图标与 tabular-nums 时间；次行为结论；末行为箭头图标 + handoff。
- 会话上下文头为属性面板式：label（12px muted）在上、value（13px + 图标）在下，只承载项目 / 工作区 / 模式。
- 行 hover 出现浅底色；无任何常驻卡片边框。

### 状态语义（圆点 + 文字）

```text
运行态：  ● 运行中(靛蓝)   ◌ 等你(中性空心)   ◌ 排队中(中性空心)
          ● 失败(红)       ● 卡住(红)         ○ 已中断(中性)
          ● 已完成(中性)   ● 已显示(中性)     ○ 空闲(中性)
裁决面：  ● 通过(绿)       ● 不通过(红)
主按钮：  靛蓝 #5E6AD2；hover 亮色加深 #4B57C8 / 暗色变亮 #828FFF；active scale(0.98)
浮层：    细描边 + 多层软投影；focus 为双层靛蓝 ring
```

Requirements:
- Badge 九个状态语义 variant（running/failed/waiting/interrupted/idle/pending/completed/displayed/stuck）视觉统一为圆点+文字，语义名与 API 不变。
- 绿/红只出现在裁决与危险事实；等你、排队、中断等状态保持中性结构信号。
- 侧栏会话状态点与 Badge 状态语义一致；侧栏四档排序（等你 → 运行中 → 静止 → 已完成折叠）不变。
- 图标统一 16px 默认尺寸、1.5px 描边。

### 字体与排版

```text
拉丁：Inter Variable（自托管子集）；UI 强调 wght 510、标题 wght 590、正文 400
全局 font-feature-settings: "cv01", "ss03"；13px 正文字距 0（负字距只用于 ≥16px 标题，本页无）
CJK：回退 PingFang SC 等系统字体，字重取最近档
```

Requirements:
- Inter Variable 以 woff2 子集（latin，约 78KB）随包自托管，OFL 1.1 license 入库，无 CDN 依赖。
- 字体未加载完成时回退系统栈，布局不抖动（font-display: swap）。
