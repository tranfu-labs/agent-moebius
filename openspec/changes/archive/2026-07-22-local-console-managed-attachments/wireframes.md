# 线框：local-console-managed-attachments

## pages/console.md

基线：`docs/wireframes/pages/console.md`。该文件已注明新会话主内容区由 `docs/product/pages/main-conversation.md#页面结构` 接管；本 change 沿用旧文件的双栏外壳，以页面 PRD 为冲突时的上位产品事实源。

归档回流目标：`docs/product/pages/main-conversation.md#页面结构`。`pages/console.md` 只标识历史线框基线，不表示要向已被接管的旧 Wireframe 写入新事实。

### 默认输入框

```text
┌──────────────────────────────────────────────────────────┐
│ ▱ 项目 A   默认工作空间   main   ◇ 开发团队 ▾           │
│ 说点什么，或 @ 一个成员…                          ＋  ↑ │
└──────────────────────────────────────────────────────────┘
                                                        ↑  ↑
                                               添加附件 ┘  └ 发送
```

「＋」位于输入框右下角，与发送按钮组成同一操作组；点击「＋」打开系统文件选择器。整个 composer 是文件拖入目标，textarea 聚焦时可粘贴剪贴板图片。

### 有附件的草稿

```text
┌──────────────────────────────────────────────────────────┐
│ ▱ 项目 A   默认工作空间   main   ◇ 开发团队 ▾           │
│ ┌──────────────┐ ┌──────────────────────────────┐       │
│ │  图片缩略图  │ │ ▤ 需求说明.pdf               │       │
│ │ 页面截图.png ×│ │   PDF · 2.4 MB          ×   │       │
│ └──────────────┘ └──────────────────────────────┘       │
│ 请对照截图和说明检查这个页面                       ＋  ↑ │
└──────────────────────────────────────────────────────────┘
```

图片项固定在可辨认但不抢正文的缩略图尺寸；普通文件卡片不渲染内容预览。文件名过长单行截断，完整名通过 accessible name/悬停可得。

### 准备中与失败

```text
准备中
┌──────────────┐
│ 图片占位     │
│ 上传中…    × │          发送禁用
└──────────────┘

失败
┌────────────────────────────────┐
│ ⚠ 需求说明.pdf 没有准备好      │
│ 磁盘空间不足      [重试] [移除]│  发送禁用
└────────────────────────────────┘
```

失败只附着在对应附件，不清空正文或其他附件；状态不能只靠颜色表达。

### 发送后的同一条消息

```text
│ 你                                                       │
│ 请对照截图和说明检查这个页面                             │
│ ┌──────────────┐ ┌──────────────────────────────┐      │
│ │  图片缩略图  │ │ ▤ 需求说明.pdf               │      │
│ │ 页面截图.png │ │   PDF · 2.4 MB               │      │
│ └──────────────┘ └──────────────────────────────┘      │
```

附件位于用户消息正文之后、下一条时间线记录之前；没有正文时从附件开始，不制造“已上传文件”之类的伪正文。

### 窄窗口

```text
┌──────────────────────────────┐
│ ┌──────────┐ ┌──────────┐   │
│ │ 图片  ×  │ │ 图片  ×  │   │
│ └──────────┘ └──────────┘   │
│ ┌────────────────────────┐   │
│ │ ▤ 需求说明.pdf      × │   │
│ └────────────────────────┘   │
│ 继续输入…              ＋ ↑ │
└──────────────────────────────┘
```

附件区自动换行，普通文件卡片最多占一行可用宽度；不产生页面级横向滚动。
