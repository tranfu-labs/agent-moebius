## pages/main-conversation.md

### 主理人与多个专业 Agent 同时运行

```text
│ (经) 开发经理                                               │
│ 正在判断是否需要调整现有派工。                              │
│                                        [完整输出 →]         │
│                                                             │
│ (开) 开发                                                   │
│ 正在修改 runtime。                      [完整输出 →] [停下] │
│                                                             │
│ (测) 测试                                                   │
│ 正在补并发测试。                         [完整输出 →] [停下] │
│ ─────────────────────────────────────────────────────────── │
│   ┌ 待发射给主理人 ─────────────────────────────────────┐  │
│   │ 1 先别改 GitHub mode                                │  │
│   │ 2 结束后再跑一次桌面验收                            │  │
│   └──────────────────────────────────────────────────────┘  │
│   ┌──────────────────────────────────────────────────────┐  │
│   │ 继续告诉主理人…                            ＋  ↑  ⏹ │  │
│   └──────────────────────────────────────────────────────┘  │
```

### 只有专业 Agent 运行

```text
│ (开) 开发                                                   │
│ 正在修改 runtime。                      [完整输出 →] [停下] │
│ ─────────────────────────────────────────────────────────── │
│   ┌──────────────────────────────────────────────────────┐  │
│   │ 这时发消息会立即给主理人                    ＋  ↑   │  │
│   └──────────────────────────────────────────────────────┘  │
```

### 精确停止

```text
点击「开发」行的 [停下]
  → 只中断开发 run
  → 测试 run 与主理人 run 继续
  → 不触发主理人待发射消息

点击 composer 的 [⏹]
  → 只中断主理人 run
  → 专业 Agent run 继续
  → 主理人终态确认后发射最早一条 pending
```
