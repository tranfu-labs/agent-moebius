04机制

流水线为何不出错

「agent 互相对话，不会越传越错吗？」——会，如果靠 agent 小心。所以我们靠工位。

STATION 01

分工

每个 agent 只做一类事。复杂任务先被调度 agent 拆成一串简单任务再上线——简单任务，错不到哪里去。

STATION 02

标准化交接

agent 之间用结构化格式交接，不靠"感觉传话"。每一次交接都有记录，可追溯、可回放。

STATION 03

质检

审察 agent 把守出口，不合格的交付原路退回重做。你验收的，永远是过了检的结果。
