# Landing Page Wireframe（Moebius 上线版）

当前唯一官网是 `sites/marketeam/index.html`。它是无构建的中文单文件静态页，以六板块连续问题链解释“可信的自动才稀缺”，而不是展示后台、工作流编辑器或未经验证的质量成绩。

## 窄屏态

```text
┌───────────────────────────┐
│ [图标] MOEBIUS    [开始]   │
├───────────────────────────┤
│ 把你在用的 AI，            │
│ 变成一个会自己推进的团队。  │
│                           │
│ 接手                       │
│  ↓                        │
│ 处理 ← 打回                │
│  ↓                        │
│ 复核 → 交付                │
├───────────────────────────┤
│ 为什么总要你盯             │
│ [拍 1 卡片]                │
│ [拍 2 卡片]                │
│ [拍 3 卡片]                │
│ [拍 4 卡片]                │
├───────────────────────────┤
│ 角色移交（纵向）            │
├───────────────────────────┤
│ 先对齐                     │
│ 再验收                     │
│ 打回 → 重做 → 再验         │
├───────────────────────────┤
│ 过程底单                   │
├───────────────────────────┤
│ Apple Silicon + CLI 检查    │
└───────────────────────────┘
```

窄屏必须保持纵向阅读和无页面级横向滚动；桌面粘性五拍在约 760px 以下改为顺序卡片。循环动效仅在视口可见时运行，`prefers-reduced-motion` 下退化为静态终态。
